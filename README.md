# 📊 IAILGO: Mistake Tracking & Analysis System

![Deployment Status](https://github.com/madanvelamuri/IAIL_GO/actions/workflows/deploy.yml/badge.svg)

**IAILGO** is a professional Quality Control (QC) analytics dashboard designed to monitor, document, and analyze processing errors within claim management workflows. Built with a focus on data visualization and evidence-based tracking.

---

## 🚀 Key Features

* **📈 Real-Time Analytics:** Interactive KPI cards and charts (Line & Bar) for instant insight into error volume.
* **🔍 Smart Filtering:** Search by **Employee**, **Date Range**, or **Mistake Type** with automatic pagination.
* **🖼️ Evidence Inspector:** High-detail image viewer with **Zoom In/Out** and **Click-to-Drag** functionality.
* **📥 CSV Export:** One-click data portability pre-formatted for Excel/Google Sheets.
* **📱 Responsive Design:** Modern, clean UI built with Tailwind CSS for all device sizes.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Charts:** Chart.js, React-Chartjs-2
- **Icons:** Lucide-React
- **Animation:** React CountUp
- **Backend:** Node.js (Hosted on Railway)
- **Deployment:** Vercel via GitHub Actions
